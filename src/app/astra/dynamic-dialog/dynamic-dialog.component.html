Dynamic Dialog

Dialogs can be created dynamically with any component as the content using a DialogService. See official documentation.

Neutral
const ref = this.dialogService.open(DynamicDialogContentComponent, { header: 'Pop-up title', footer: 'Dynamic Dialog Footer', width: '70vw', }); ref.onClose.pipe(take(1)).subscribe((res) => { // do something based on res console.log('dynamic dialog result', res); });
Within Day
const ref = this.dialogService.open(DynamicDialogContentComponent, { header: 'Pop-up title', footer: 'Dynamic Dialog Footer', width: '70vw', styleClass: 'flx-within-day', }); ref.onClose.pipe(take(1)).subscribe((res) => { // do something based on res console.log('dynamic dialog result', res); });
