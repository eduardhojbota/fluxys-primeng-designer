Fluxys PrimeNG Theme

Welcome to the demo website dedicated to show how the default PrimeNG components look like with the new theme created specifically for Fluxys.

This theme is based on the Astra design created by the UX team and documented in Figma.

Use the left menu to navigate to the different PrimeNG components and see how they look like.

On each component page, there is, at least, the Default Astra Styling that show how the PrimeNG component looks like with the default Astra Design System.
Sometimes, the components has extra styling defined for GSmart applications like the Within Day and Day Ahead colors. In that case, there is an additional panel showing the custom styling for the PrimeNG component.
