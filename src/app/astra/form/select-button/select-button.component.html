SelectButton

SelectButton is a form component to choose a value from a list of options using button elements. See official documentation.

Single Selection
<p-selectButton styleClass="flx-select-button-secondary" [options]="stateOptions" [(ngModel)]="value" optionLabel="label" optionValue="value"></p-selectButton> this.stateOptions = [{ label: 'Off', value: 'off' }, { label: 'On', value: 'on' }];
Multiple Selection
<p-selectButton styleClass="flx-select-button-secondary" [options]="paymentOptions" [(ngModel)]="value" multiple="multiple" optionLabel="name" optionValue="value" optionDisabled="disabled"></p-selectButton> this.paymentOptions = [ { name: 'Option 1', value: 1, disabled: false }, { name: 'Option 2', value: 2, disabled: true }, { name: 'Option 3', value: 3, disabled: false } ];
